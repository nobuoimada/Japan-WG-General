   - その他資料  
   OpenChain Japan WGでの活動とは関係なく作成したものを各社が会合等で紹介する場合は、著作権法の通り、各社がライセンス条件を定めて下さい。配布を許諾する場合は、ライセンス条件を明記して提供して下さい。
